# 💫 Botics:
Backend para la gestión de los laboratorios de informática ESFOT, este presenta una serie de endpoints que permiten gestionar la información referente a los laboratorios de informática de la Escuela de Formación de Tecnólogos, estos están descritos mediante una API presentada con SwaggerUI y desplegada como una aplicación contenida con Docker Y Azure Apps
